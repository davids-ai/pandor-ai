# Prueba Blog (Angular scaffold)

Proyecto scaffold mínimo de Angular con estructura para un blog (componentes, rutas, servicio mock).

Requisitos:
- Node.js (>=18 recomendable)
- npm

Instalación y ejecución (PowerShell):
```powershell
cd C:\Users\HELL\Desktop\prueba
npm install
npx ng serve --open
```

Notas:
- El proyecto contiene datos mock en `src/app/services/post.service.ts`.
- Para usar Angular CLI localmente puedes ejecutar `npx ng` o instalar `@angular/cli` globalmente.
